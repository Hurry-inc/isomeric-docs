description: "This endpoint returns the status of an `/extract` request"
---

### Headers

<ParamField
  header="Authorization"
  required="true"
  initialValue="Bearer iso_abc123"
>
  Use your API key to authenticate the request.
</ParamField>

### URL Path Params

<ParamField body=":id" type="string" required="true">
  The request (job) ID returned from your call to `/extract`
</ParamField>

### Response

<ResponseField name="status" type="object">
  The status of the requested request (job). Values will be one of `completed`,
  `inflight`, `failed`. In flight requests are still being processed.
</ResponseField>

<ResponseField name="data" type="object">

The data returned from the extraction request, which is identical to that from `/extract`
