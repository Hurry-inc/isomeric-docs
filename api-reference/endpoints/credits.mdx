---
title: "Check Credits"
api: "GET https://api-preview.isomeric.ai/credits"
description: "This endpoint returns the credit balance for the authenticating API key"
---

### Headers

<ParamField
  header="Authorization"
  required="true"
  initialValue="Bearer iso_abc123"
>
  Use your API key to authenticate the request.
</ParamField>

### Response

<ResponseField name="key" type="string">
  Echos back the API key used to authenticate.
</ResponseField>

<ResponseField name="remaining" type="number">
  The number of credits left.
</ResponseField>

<ResponseField name="buyMoreCredits" type="object">

Payment links to purchase more credit packs.

<Expandable title="Toggle object">

<ResponseField name="Plan Name" type="string">
  Each payment link will be returned as a key-value pair, where the key is the
  package name and the value is the URL to purchase that pack.
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl -s -X GET 'https://api-preview.isomeric.ai/credits' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer <token>'
```

</RequestExample>
